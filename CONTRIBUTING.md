# Contributing to Overpowered Toys

Thank you for your interest in contributing!

## How to contribute

- Fork the repository
- Create a new branch (`feature/your-feature-name`)
- Make your changes
- Test your changes in RimWorld
- Open a Pull Request

## Code Style

- Follow RimWorld XML structure conventions
- Keep naming consistent
- Do not modify other features without discussion

## Reporting Bugs

Please use the Bug Report template and include:

- RimWorld version
- Mod version
- Player.log file
- Steps to reproduce

## Feature Requests

Please use the Feature Request template and describe:

- What problem it solves
- Why it should be added
- Optional balance suggestions
